    <h1 align="center">Yuto Shima Portfolio</h1>

個人ポートフォリオサイトです。[Next.js](https://nextjs.org/)、[Tailwind CSS](https://tailwindcss.com/)、[Contentlayer](https://www.contentlayer.dev/) を使用して構築し、[GitHub Pages](https://pages.github.com/)にデプロイしています。

<br/>

## ローカルでの実行方法

```sh
git clone https://github.com/yutoshima/my-profile-app.git
cd my-profile-app
npm install
npm run dev
```

## 機能

- 個人情報と自己紹介の表示
- プロジェクト一覧の表示
- コンタクト情報の表示
- レスポンシブデザイン
- アニメーション効果
- 日本語対応

## 技術スタック

- **フレームワーク**: Next.js
- **スタイリング**: Tailwind CSS
- **コンテンツ管理**: Contentlayer
- **デプロイ**: GitHub Pages
